## THE FRAMING SHIFT

The product doesn't change. The demo doesn't change. The dashboard doesn't change.
What changes is the VOCABULARY — the specific words you choose when narrating.

Risk & Compliance judges are listening for:
- "monitoring" / "safeguards" / "automated controls"
- "risk detection" / "risk scoring" / "risk thresholds"
- "circuit breaker" / "kill switch" / "emergency halt"
- "audit trail" / "forensic reporting" / "compliance"
- "protocol safety" / "systemic risk" / "fails closed"

CRE & AI judges are listening for:
- "AI-in-the-loop" / "autonomous agents" / "AI consensus"
- "CRE workflow" / "orchestration layer" / "verifiable execution"
- "Confidential Compute" / "TEE" / "attested verdict"

The v4.1 script hits BOTH vocabularies but leads with risk/compliance language.
Below is every changed line, marked with what changed and why.
